Online Library Management System
An Online Library Management System built using EJB, JEE, JPA, JSP, Servlets, GlassFish, MySQL, and Apache Java with Ant. This system allows students to view available books, borrow them, and allows admins to manage the library catalog, track borrowed books, and update return dates.

Features
For Students:
View Available Books: Browse and search through the available books.

Borrow Books: Borrow a book for a specified time period.

View Borrowed Books: View a list of books borrowed by the student, along with due dates.

For Admins:
Add Books: Add new books to the library catalog.

Update Book Information: Modify details for existing books (e.g., title, author, availability).

View Borrowed Books: View a list of all borrowed books and their due dates.

Update Return Dates: Update the return dates for borrowed books when returned or extended.

Technologies Used
Frontend: JSP (JavaServer Pages), HTML, CSS

Backend:

EJB (Enterprise JavaBeans) for managing business logic and transactions

JEE (Java EE) for building the web application

JPA (Java Persistence API) for interacting with the MySQL database

Servlets for handling HTTP requests and controlling user interaction

Application Server: GlassFish to deploy and run the Java EE application

Database: MySQL to store book, user, and transaction data

Build Automation: Apache ANT for building and managing the project

Installation
Prerequisites
JDK (Java Development Kit) 8 or higher

Apache Java with Ant for build management

GlassFish Server for deploying Java EE applications
